# Reborn Infinity

完全無料・超多重転生 放置RPG

## 概要
- Tier0〜Tier15の16段階転生システム
- 職業300+・スキル500+・装備1000+・地域200+
- 実績800+・称号300+・図鑑多数

## プレイ方法
`index.html`をブラウザで開くだけでプレイ可能。

## ファイル構成
- `index.html` — メインHTML
- `css/` — スタイルシート
- `js/core/` — ゲームエンジン
- `js/jobs/` — 職業システム
- `js/skills/` — スキルシステム
- `js/ui.js` — UI統合
